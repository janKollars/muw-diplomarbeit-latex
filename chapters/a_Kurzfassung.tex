\chapter*{Kurzfassung}

Kurzfassung hier einfügen. Sie sollte nicht länger als eine Seite sein. Sie soll folgendermaßen strukturiert sein:

\begin{enumerate}
    \item kurze Darstellung von Hintergrund
    \item Problemstellung
    \item Rationale
    \item Methodik
    \item Verfahren
    \item Ergebnisse
    \item Schlussfolgerungen
\end{enumerate}
